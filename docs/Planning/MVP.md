# SplitCents MVP

## MVP goal
The MVP should help a single user plan their monthly money in a simple, low-friction way. The focus is on making it easy to understand what money is coming in, what recurring bills must be paid, how much can be allocated to spending categories, and how savings goals can be supported over time.

## MVP scope
The first release should cover the following core experience:

### 1. User setup
- Create an account and sign in.
- Add a user profile with basic personal details.
- Set a monthly take-home income.

### 2. Income and bills
- Add monthly take-home pay.
- Add recurring bills with a name, amount, frequency, and due date.
- Support monthly bills by default, with weekly and annual bills as options.
- Mark bills as paid or unpaid.

### 3. Budget planning
- Create monthly budget categories such as housing, food, transport, fun, and debt payments.
- Set budget amounts as either fixed values or percentages of income.
- See how much remains in each budget category.

### 4. Savings goals
- Create savings goals with a target amount, current amount saved, target date, and monthly contribution.
- See progress toward each goal.
- Estimate how long it will take to reach the goal based on the current contribution plan.

### 5. Monthly overview
- View a simple dashboard showing income, bills, remaining money, and savings progress.
- See upcoming bills and a clear monthly summary.

## MVP user stories
- As a user, I want to enter my monthly take-home pay so I can plan my money.
- As a user, I want to add recurring bills so I know what money is already committed.
- As a user, I want to create budget categories so I can plan spending in advance.
- As a user, I want to set savings goals so I can track progress over time.
- As a user, I want to see a monthly overview so I can quickly understand my financial position.

## Out of scope for MVP
The following features are intentionally excluded from the first release:
- Bank account linking and transaction import
- Automatic categorisation of real spending data
- Advanced forecasting or “what if” simulation tools
- Shared household planning for couples
- Deep investment or retirement planning features

## MVP success criteria
The MVP can be considered successful if:
- A user can set income, bills, budgets, and savings goals in one session.
- The app presents a clear monthly summary without requiring complex setup.
- The experience is simple enough for a first-time user to understand quickly.
- The core flow works reliably for solo personal budgeting.

## Assumptions
- The product is initially focused on personal budgeting rather than shared household budgeting.
- The app will be manually entered and controlled rather than connected to external financial accounts.
- The MVP should prioritise clarity and usability over advanced financial features.
