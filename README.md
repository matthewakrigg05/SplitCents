# SplitCents

Welcome to SplitCents! A budgeting app, designed for couples and partners, but of course free to be used by individuals.

The aim of SplitCents is to provide a central space where you can plan spending, saving and investing. This is currently a fresh idea,
and so this space will be populated over the coming days, as I flesh out my ideas and intentions with this application. 

Watch this space!
